# API 使用指南

## 基本信息

- **API 地址**：`https://api.tanstudio.me`
- **接口格式**：OpenAI 兼容格式
- **支持模型**：40+ 种主流 AI 模型

---

## 快速开始

### 1. 获取 API Key

登录 [管理后台](https://api.tanstudio.me) → 令牌管理 → 创建新令牌

### 2. 测试连接

```bash
# 测试获取模型列表
curl https://api.tanstudio.me/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 配置指南

### Claude Code

在终端中设置环境变量：

```bash
# Linux/Mac
export ANTHROPIC_API_KEY="YOUR_API_KEY"
export ANTHROPIC_BASE_URL="https://api.tanstudio.me"

# Windows PowerShell
$env:ANTHROPIC_API_KEY="YOUR_API_KEY"
$env:ANTHROPIC_BASE_URL="https://api.tanstudio.me"
```

或在 `~/.claude/config.json` 中配置：

```json
{
  "apiKey": "YOUR_API_KEY",
  "baseUrl": "https://api.tanstudio.me"
}
```

---

### OpenAI SDK (Python)

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.tanstudio.me/v1"
)

response = client.chat.completions.create(
    model="mimo-v2.5-pro",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

print(response.choices[0].message.content)
```

---

### OpenAI SDK (Node.js)

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://api.tanstudio.me/v1'
});

const response = await client.chat.completions.create({
  model: 'mimo-v2.5-pro',
  messages: [
    { role: 'user', content: 'Hello!' }
  ]
});

console.log(response.choices[0].message.content);
```

### 聊天测试

```bash
# 简单聊天
curl https://api.tanstudio.me/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{

    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'
```
